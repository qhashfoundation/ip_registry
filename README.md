# ip_registry

This repository publishes blockchain-timestamped cryptographic hash records and their digital signatures (* stamp), providing tamper-evident proof of existence and integrity (hash + timestamp), attribution (signature), and a verifiable binding to the owner’s Qhash identity (public-key identity linkage) for all source code/compiled code published on  
https://github.com/qhashfoundation
